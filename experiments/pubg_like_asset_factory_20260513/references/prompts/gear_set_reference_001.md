# Gear Set Reference Prompt 001

Purpose: realistic references for `gear_set_v1`.

Prompt:

```text
Realistic modular tactical gear asset sheet, fast helmet, plate carrier, magazine pouches, backpack, gloves, knee pads, boots, radio pouch, worn fabric, polymer buckles, mud stains, edge wear, isolated objects on neutral gray studio background, multiple angles, high detail PBR material cues, survival shooter asset reference.
```

Negative prompt:

```text
fantasy, sci-fi, toy plastic, pristine clean gear, fashion catalog, readable brand logos, text labels, cartoon style
```

Acceptance:

- modules can be separated or combined into player/enemy character packets;
- shows fabric, polymer, metal, and dirt/wear material zones;
- supports close-up gear screenshots.
