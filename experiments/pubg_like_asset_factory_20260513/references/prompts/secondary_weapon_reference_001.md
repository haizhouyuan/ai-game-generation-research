# Secondary Weapon Reference Prompt 001

Purpose: realistic reference for `secondary_weapon_v1`, an additional visible weapon pickup beyond the hero rifle and sidearm.

Prompt:

```text
Realistic modern tactical compact long-gun pickup for a survival shooter, worn metal receiver, rail silhouette, simple optic, magazine, tactical sling loop, mud and scratch wear, isolated on neutral gray studio background, orthographic side view plus three-quarter view, high detail PBR material cues, game asset reference, no sci-fi styling, no exploded parts, no manufacturing diagram.
```

Negative prompt:

```text
toy plastic, pristine showroom, fantasy, neon sci-fi, blueprint, instructional cutaway, disassembled weapon, text labels, cartoon style
```

Acceptance:

- visually distinct from the hero rifle and sidearm;
- enough material-zone separation for basecolor/normal/roughness/metallic authoring;
- suitable for loot pickup, third-person held view, and background weapon rack context.
