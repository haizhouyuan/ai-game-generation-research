# Enemy Tactical Character Reference Prompt 001

Purpose: realistic reference for `enemy_tactical_v1`.

Prompt:

```text
Realistic modern tactical enemy character for rainy checkpoint scene, dark worn tactical jacket, plate carrier, helmet, gloves, knee pads, muddy boots, muted faction patch without readable text, front side back character sheet, neutral gray background, realistic fabric weave, scuffed armor plates, wet material highlights, game asset reference, grounded proportions.
```

Negative prompt:

```text
fantasy armor, sci-fi exosuit, superhero, cartoon, clean cosplay, oversized weapons, readable logos, text labels, extreme gore
```

Acceptance:

- distinct from player silhouette while staying realistic;
- enough front/side/back information for rigged character or Blender assembly;
- material zones visible for fabric, polymer, metal, boots, and gloves.
