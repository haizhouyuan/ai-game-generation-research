# Hunyuan Shape Demo 001 Reference

This packet is the first local Hunyuan3D-2.1 shape-generation proof for the PUBG-like full asset factory goal.

- Input image: `source/images/input_demo.png`
- Input origin: official Hunyuan3D-2.1 repository demo image copied from the HomePC repo clone
- Model route: `tencent/Hunyuan3D-2.1` standalone shape pipeline
- Host: HomePC, `CUDA_VISIBLE_DEVICES=1`
- Output: `model/raw.glb`

This is not a final game asset. It is accepted only as an environment and pipeline proof that local Hunyuan3D-2.1 shape generation can produce a GLB without using proxy traffic for the model download.
