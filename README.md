# AI Game Generation Research

This repo studies AI-assisted 3D game generation from concept to playable prototype. It is research-first: tool discovery, pipeline mapping, local reproducibility, capability limits, and prototype path selection.

## Current Focus

- Concept and design generation.
- Image-to-3D and text-to-3D assets.
- Materials, rigging, animation, and environment assembly.
- Godot, Unity, and Unreal import paths.
- Browser or desktop automation where useful.
- Evaluation of what can actually be reproduced locally.

## Boundary

No private child data, private screenshots, account sessions, or hosted uploads are used unless explicitly approved by the user.
